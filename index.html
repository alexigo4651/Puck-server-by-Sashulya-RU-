🚀 Инфраструктура серверов Puck

Установка и настройка сервера (Обновлено для Build 202)

Требования:
  • Сервер на базе Debian (Debian 12 bookworm) или Ubuntu с SSH-доступом.
  • Root-доступ (все команды выполняются от имени пользователя root).
  • Минимум 2 ГБ ОЗУ (каждый сервер Puck потребляет ~300 МБ).
  • Базовые знания командной строки.
Примечание: Для работы сервера будет создан отдельный пользователь `puckd`.

Шаг 1: Обновление системы

apt update && apt upgrade -y

Шаг 2: Установка зависимостей и SteamCMD

Установите необходимые инструменты и подготовьте SteamCMD:

apt install software-properties-common
echo "deb http://deb.debian.org/debian bookworm main contrib non-free" > /etc/apt/sources.list.d/nonfree.list
dpkg --add-architecture i386
apt update
apt install steamcmd

Шаг 3: Создание папки для загрузки сервера

mkdir -p /srv/puck-download

Шаг 4: Загрузка файлов сервера Puck

Используйте SteamCMD для загрузки файлов выделенного сервера (App ID 3481440):

/usr/games/steamcmd +force_install_dir /srv/puck-download +login anonymous +app_update 3481440 +quit

Шаг 5: Создание системного пользователя

Создайте выделенного пользователя для безопасного запуска сервера:

adduser --system --no-create-home --group puckd

Шаг 6: Настройка первого экземпляра сервера

Создайте директорию для первого сервера и назначьте права:

mkdir -p /srv/puck/server1
cp -r /srv/puck-download/* /srv/puck/server1/
chown -R puckd:puckd /srv/puck/server1
chmod +x /srv/puck/server1/start_server.sh
chmod +x /srv/puck/server1/Puck.x86_64

Откройте файл конфигурации:

nano /srv/puck/server1/config.json

Важные настройки:
• Name: Имя сервера в браузере. Поддерживает Unity Rich Text (цвета и размеры), но не злоупотребляйте этим.
• Port: Порт сервера (7777 по умолчанию). Его нужно пробросить в роутере или включить UPnP.
• targetFrameRate: Фактический FPS сервера (рекомендуется ставить на 10-20 выше тикрейта).
• serverTickRate: Частота обновления сервера. Рекомендуется 240-360Hz+, если позволяет железо.
• clientTickRate: Частота отправки данных от клиентов (влияет на инпут-лаг). Можно ставить равной серверному тикрейту.
• Mods: В примере добавлены Toaster's Crispy Shadows (клиентский) и GAFURIX' Custom MOTD (серверный).

Шаг 8: Создание службы systemd

Создайте юнит для удобного управления сервером:

nano /etc/systemd/system/puck@.service

Вставьте следующее содержимое:

[Unit]
Description=Puck Dedicated Server - %i
After=network.target

[Service]
WorkingDirectory=/srv/puck/%i
User=puckd
ExecStart=/srv/puck/%i/start_server.sh --serverConfigurationPath /srv/puck/%i/config.json
Restart=on-failure

[Install]
WantedBy=multi-user.target

    mv "$TMP_MERGE" "$MASTER_DB"
    rm -f "$INCOMING_DIR"/*.csv

cat << 'EOF' > /srv/puck/server1/Logs/update_steam_db.sh
#!/bin/bash

LOG_DIR="/srv/puck/server1/Logs"
DB_FILE="$LOG_DIR/steam_db.csv"
TMP_IPS="/tmp/steam_ips.txt"
TMP_NICKS="/tmp/steam_nicks.txt"

if [ ! -f "$DB_FILE" ]; then
    echo "SteamID;IP_Addresses;Nicknames" > "$DB_FILE"
fi

cat "$LOG_DIR"/ip_logger_*.ndjson 2>/dev/null | \
jq -R -r 'fromjson? | select(.steam_id != null and .ip != null) | "\(.steam_id)\t\(.ip)"' | \
awk '{
    split(a[$1], ips, ",")
    found = 0
    for (j in ips) if (ips[j] == $2) found = 1
    if (!found) a[$1] = a[$1] ? a[$1] "," $2 : $2
} END { for (k in a) print k "\t" a[k] }' > "$TMP_IPS"

gawk '
{
    sid=""; nick=""
    if (match($0, /Steam Id ([0-9]{17}).*\(([^)]+)\)/, m)) {
        sid = m[1]; nick = m[2]
    } 
    else if (match($0, /\[UIChat\] \[[^\]]+\] (.*) \([0-9]+\) \[([0-9]{17})\]:/, c)) {
        nick = c[1]; sid = c[2]
    }

    if (sid != "" && nick != "") {
        gsub(/^[ \t]+|[ \t]+$/, "", nick)
        if (nick ~ /^[0-9]{17}$/) next
        
        split(nicks[sid], ns, ",")
        found = 0
        for (i in ns) if (ns[i] == nick) found = 1
        if (!found) nicks[sid] = nicks[sid] ? nicks[sid] "," nick : nick
    }
} END { for (k in nicks) print k "\t" nicks[k] }' "$LOG_DIR"/Puck*.log 2>/dev/null > "$TMP_NICKS"

awk '
BEGIN { FS="[;\t]"; OFS=";" }
function merge(old, new,    arr, k, res, uniq) {
    split(old "," new, arr, ",")
    for (k in arr) if (arr[k] != "" && arr[k] != "n/a") uniq[arr[k]] = 1
    res = ""
    for (k in uniq) res = (res == "") ? k : res "," k
    return (res == "" ? "n/a" : res)
}
{
    if ($1 == "SteamID") next
    sid = $1
    if (FILENAME == db) { i[sid]=$2; n[sid]=$3 }
    else if (FILENAME == t_i) i[sid] = merge(i[sid], $2)
    else if (FILENAME == t_n) n[sid] = merge(n[sid], $2)
}
END {
    print "SteamID;IP_Addresses;Nicknames"
    for (sid in i) sids[sid]=1; for (sid in n) sids[sid]=1
    for (sid in sids) {
        print sid, (i[sid]?i[sid]:"n/a"), (n[sid]?n[sid]:"n/a")
    }
}' db="$DB_FILE" t_i="$TMP_IPS" t_n="$TMP_NICKS" "$DB_FILE" "$TMP_IPS" "$TMP_NICKS" > "${DB_FILE}.tmp"

mv "${DB_FILE}.tmp" "$DB_FILE"
rm -f "$TMP_IPS" "$TMP_NICKS"
EOF
chmod +x /srv/puck/server1/Logs/update_steam_db.sh

1. Установка Python и запуск API на Мастере

cat << 'EOF' > /root/puck-api.py
import subprocess
import csv
import os
import re
import json
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware
from pydantic import BaseModel

app = FastAPI()

app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],
    allow_methods=["*"],
    allow_headers=["*"],
)

# Пути к файлам и директориям
SERVERS_FILE = "/root/servers.json"
SSH_DIR = "/root/.ssh/puck_keys"
DB_PATH = "/srv/puck/server1/Logs/steam_db.csv"
BAN_FILE = "/srv/puck/server1/banned_steam_ids.json"
SYNC_SCRIPT = "/root/sync_bans.sh"
MANUAL_BANS_TXT = "/srv/puck/server1/manual_bans.txt"

# Создаем папку для ключей, если ее нет
os.makedirs(SSH_DIR, exist_ok=True)

class ServerModel(BaseModel):
    id: str
    name: str
    ip: str
    ssh_ip: str = None
    ssh_key: str = None

def get_servers():
    if not os.path.exists(SERVERS_FILE):
        data = {
            "master": {
                "name": "Master Server", 
                "ip": "193.42.124.91", 
                "ssh_ip": "127.0.0.1"
            }
        }
        with open(SERVERS_FILE, "w", encoding="utf-8") as f: 
            json.dump(data, f, indent=4)
            
    with open(SERVERS_FILE, "r", encoding="utf-8") as f: 
        return json.load(f)

def save_servers(data):
    with open(SERVERS_FILE, "w", encoding="utf-8") as f: 
        json.dump(data, f, indent=4)

def run_cmd(server_id: str, cmd: str):
    servers = get_servers()
    srv = servers.get(server_id)
    if not srv: 
        return "", "Server not found in config", 1
    
    target_ip = srv.get("ssh_ip")
    if not target_ip:
        target_ip = srv["ip"]
    
    if server_id == "master" or target_ip in ["127.0.0.1", "localhost"]:
        full_cmd = cmd
    else:
        key_path = f"{SSH_DIR}/{server_id}"
        full_cmd = f"ssh -i {key_path} -o StrictHostKeyChecking=no -o ConnectTimeout=5 root@{target_ip} '{cmd}'"
    
    try:
        res = subprocess.run(full_cmd, shell=True, capture_output=True, text=True, timeout=15)
        return res.stdout, res.stderr, res.returncode
    except Exception as e:
        return "", str(e), 1

@app.get("/db/banned-ids")
async def get_banned_ids():
    if os.path.exists(BAN_FILE):
        with open(BAN_FILE, "r") as f:
            try:
                return json.load(f)
            except:
                return []
    return []
    
@app.post("/server/steam-ban")
async def steam_ban(data: dict):
    sid = data.get('steam_id')
    action = data.get('action')
    try:
        bans = []
        if os.path.exists(BAN_FILE):
            with open(BAN_FILE, "r") as f:
                content = f.read()
                if content: bans = json.loads(content)
        if action == "ban":
            if sid not in bans: bans.append(sid)
        elif action == "unban":
            if sid in bans: bans.remove(sid)
        with open(BAN_FILE, "w") as f:
            json.dump(bans, f, indent=4)
        subprocess.run(f"bash {SYNC_SCRIPT}", shell=True)
        return {"status": "success", "message": f"ID {sid} {action}ed"}
    except Exception as e:
        return {"status": "error", "message": str(e)}

@app.post("/servers/save")
def api_save_server(srv: ServerModel):
    servers = get_servers()
    if srv.ssh_key and srv.ssh_key.strip():
        key_path = f"{SSH_DIR}/{srv.id}"
        with open(key_path, "w") as f:
            f.write(srv.ssh_key.strip() + "\n")
        os.chmod(key_path, 0o600)
    servers[srv.id] = {
        "name": srv.name, 
        "ip": srv.ip, 
        "ssh_ip": srv.ssh_ip or srv.ip
    }
    save_servers(servers)
    return {"status": "success"}

@app.delete("/servers/{server_id}")
def api_del_server(server_id: str):
    if server_id == "master":
        return {"error": "Cannot delete master server."}
    servers = get_servers()
    if server_id in servers:
        del servers[server_id]
        save_servers(servers)
        key_path = f"{SSH_DIR}/{server_id}"
        if os.path.exists(key_path): 
            os.remove(key_path)
        return {"status": "success"}
    return {"error": "Server not found."}

@app.get("/server/status")
def get_status(server_id: str):
    _, _, code = run_cmd(server_id, "systemctl is-active puck@server1 --quiet")
    is_online = (code == 0)
    player_count = 0
    if is_online:
        cmd = 'tail -n 2500 /srv/puck/server1/Logs/Puck.log | grep -E "Added clientId|Client disconnected|UIChat|subscription from|Pre-Connection approved"'
        out_on, _, _ = run_cmd(server_id, cmd)
        players = {}
        for line in out_on.splitlines():
            join = re.search(r"Added clientId (\d+).*?linked to Steam Id \d+ \((.*?)\)", line)
            if join:
                players[join.group(1)] = join.group(2)
            pre = re.search(r"Pre-Connection approved for (\d+)", line)
            if pre:
                players[pre.group(1)] = "Connecting..."
            sub = re.search(r"Received subscription from (.*?) \((\d+)\)", line)
            if sub:
                players[sub.group(2)] = sub.group(1)
            chat = re.search(r"\[UIChat\] \[.*?\] (.*?) \((\d+)\) \[(\d+)\]", line)
            if chat:
                players[chat.group(2)] = chat.group(1)
            leave = re.search(r"Client disconnected \((\d+)\)", line)
            if leave:
                p_id = leave.group(1)
                if p_id in players:
                    del players[p_id]
        player_count = len(players)
    return {"online": is_online, "players": player_count}

@app.get("/server/live-logs")
def get_live_logs(server_id: str):
    db_path = "/srv/puck/server1/Logs/steam_db.csv"
    nicks = {}
    try:
        with open(db_path, "r", encoding="utf-8") as f:

    header = f"{'SteamID':<20} | {'Time':<10} | {'Nickname':<25} | {'IP':<18} | {'Status'}\\n"

    formatted_out = ""

        if len(parts) == 4:

            name = nicks.get(sid, "New Player")
            formatted_out += f"{sid:<20} | {short_time:<10} | {name[:25]:<25} | {ip:<18} | {dec}\\n"
    return {"output": header + formatted_out}

        join = re.search(r"Added clientId (\d+) linked to Steam Id (\d+) \((.*?)\)", line)
        if join:
            players[join.group(1)] = {"steam": join.group(2), "name": join.group(3)}
        pre = re.search(r"Pre-Connection approved for (\d+) \((\d+)\)", line)
        if pre:
            cid, sid = pre.group(1), pre.group(2)
            if cid not in players:
                players[cid] = {"steam": sid, "name": "Connecting..."}
            else:
                players[cid]["steam"] = sid
        sub = re.search(r"Received subscription from (.*?) \((\d+)\)", line)
        if sub:
            name, cid = sub.group(1), sub.group(2)
            if cid not in players:
                players[cid] = {"steam": "N/A", "name": name}
            else:
                players[cid]["name"] = name
        chat = re.search(r"\[UIChat\] \[.*?\] (.*?) \((\d+)\) \[(\d+)\]", line)
        if chat:
            players[chat.group(2)] = {"steam": chat.group(3), "name": chat.group(1)}
        leave = re.search(r"Client disconnected \((\d+)\)", line)
        if leave and leave.group(1) in players:
            del players[leave.group(1)]
    return {"count": len(players), "list": list(players.values())}

@app.get("/db")
def get_db():
    if not os.path.exists(DB_PATH): return []
    players = []
    with open(DB_PATH, mode='r', encoding='utf-8') as f:
        reader = csv.DictReader(f, delimiter=';')
        for row in reader: players.append(row)
    return players

if __name__ == "__main__":

Данную команду вводим на клиенте и меняем MASTER_IP на IP своего Мастер сервера 
cat << 'EOF' > /root/send_to_master_bans.sh
#!/bin/bash

# --- НАСТРОЙКИ ---
LOCAL_BANS="/srv/puck/server1/banned_steam_ids.json"
MASTER_IP="193.42.124.91"
MASTER_DIR="/tmp/incoming_bans"
HOSTNAME=$(hostname)
TMP_DOWNLOAD="/tmp/master_bans.json"
TMP_MERGE="/tmp/local_merged_bans.json"

echo "--- Синхронизация с Мастером ($MASTER_IP) ---"

# 1. Отправляем свои баны на мастер (с таймаутом 5 сек)
scp -o ConnectTimeout=5 "$LOCAL_BANS" root@"$MASTER_IP":"$MASTER_DIR"/"${HOSTNAME}_bans.json"

# 2. Скачиваем общую базу с мастера
scp -o ConnectTimeout=5 root@"$MASTER_IP":"/srv/puck/server1/banned_steam_ids.json" "$TMP_DOWNLOAD"

# 3. Умное слияние (только если файл скачался и он не пустой)
if [ -s "$TMP_DOWNLOAD" ]; then
    # Склеиваем локальный файл и файл мастера, убираем дубликаты через jq
    jq -s 'add | unique' "$LOCAL_BANS" "$TMP_DOWNLOAD" > "$TMP_MERGE"
    
    if [ -s "$TMP_MERGE" ]; then
        mv "$TMP_MERGE" "$LOCAL_BANS"
        # Даем права, чтобы игровой сервер мог читать и изменять файл
        chmod 666 "$LOCAL_BANS"
        echo "Успех: Базы объединены, файл обновлен."
    fi
    rm -f "$TMP_DOWNLOAD"
else
    echo "Ошибка: Не удалось получить данные с Мастера или файл пуст."
fi
EOF

chmod +x /root/send_to_master_bans.sh

        echo "Успешно: $IP обновлен."
    else
        echo "ОШИБКА: Не удалось отправить на $IP"

echo "--- Синхронизация завершена ---"

TMP_MERGE="/tmp/merged_bans.json"

# Создаем папку, если её нет
mkdir -p "$INCOMING_DIR"

echo "--- Запуск слияния баз ---"

# Проверяем, есть ли файлы от клиентов
if ls "$INCOMING_DIR"/*.json 1> /dev/null 2>&1; then
    # Склеиваем базу мастера со всеми присланными файлами через jq
    jq -s 'add | unique' "$MASTER_BANS" "$INCOMING_DIR"/*.json > "$TMP_MERGE"

    if [ -s "$TMP_MERGE" ]; then
        mv "$TMP_MERGE" "$MASTER_BANS"
        rm -f "$INCOMING_DIR"/*.json
        echo "Баны успешно склеены и очищены."
    else
        echo "Ошибка: получился пустой файл, отмена."
    fi
else
    echo "Нет новых файлов для слияния."
fi
EOF

chmod +x /root/merge_bans.sh

    "1 ip"
