🚀 Инфраструктура серверов Puck

Установка и настройка сервера (Обновлено для Build 202)

Требования:
  • Сервер на базе Debian (Debian 12 bookworm) или Ubuntu с SSH-доступом.
  • Root-доступ (все команды выполняются от имени пользователя root).
  • Минимум 2 ГБ ОЗУ (каждый сервер Puck потребляет ~300 МБ).
  • Базовые знания командной строки.
Примечание: Для работы сервера будет создан отдельный пользователь `puckd`.

Шаг 1: Обновление системы

apt update && apt upgrade -y

Шаг 2: Установка зависимостей и SteamCMD

Установите необходимые инструменты и подготовьте SteamCMD:

apt install software-properties-common
echo "deb http://deb.debian.org/debian bookworm main contrib non-free" > /etc/apt/sources.list.d/nonfree.list
dpkg --add-architecture i386
apt update
apt install steamcmd

Шаг 3: Создание папки для загрузки сервера

mkdir -p /srv/puck-download

Шаг 4: Загрузка файлов сервера Puck

Используйте SteamCMD для загрузки файлов выделенного сервера (App ID 3481440):

/usr/games/steamcmd +force_install_dir /srv/puck-download +login anonymous +app_update 3481440 +quit

Шаг 5: Создание системного пользователя

Создайте выделенного пользователя для безопасного запуска сервера:

adduser --system --no-create-home --group puckd

Шаг 6: Настройка первого экземпляра сервера

Создайте директорию для первого сервера и назначьте права:

mkdir -p /srv/puck/server1
cp -r /srv/puck-download/* /srv/puck/server1/
chown -R puckd:puckd /srv/puck/server1
chmod +x /srv/puck/server1/start_server.sh
chmod +x /srv/puck/server1/Puck.x86_64

Шаг 7: Создание конфигурации сервера

Откройте файл конфигурации:

nano /srv/puck/server1/config.json

Вставьте и измените по необходимости следующий код:

{
  "port": 7777,
  "pingPort": 7778,
  "name": "My Dedicated Puck Server",
  "maxPlayers": 10,
  "password": "",
  "voip": false,
  "isPublic": true,
  "adminSteamIds": [],
  "reloadBannedSteamIds": false,
  "usePuckBannedSteamIds": true,
  "printMetrics": true,
  "kickTimeout": 300,
  "sleepTimeout": 60,
  "joinMidMatchDelay": 10,
  "targetFrameRate": 120,
  "serverTickRate": 100,
  "clientTickRate": 200,
  "startPaused": false,
  "allowVoting": true,
  "phaseDurationMap": {
    "Warmup": 600,
    "FaceOff": 3,
    "Playing": 300,
    "BlueScore": 5,
    "RedScore": 5,
    "Replay": 10,
    "PeriodOver": 15,
    "GameOver": 15
  },
  "mods": [
    {
      "id": 3493628417,
      "enabled": false,
      "clientRequired": true
    },
    {
      "id": 3491056040,
      "enabled": true,
      "clientRequired": false
    }
  ]
}
Важные настройки:
• Name: Имя сервера в браузере. Поддерживает Unity Rich Text (цвета и размеры), но не злоупотребляйте этим.
• Port: Порт сервера (7777 по умолчанию). Его нужно пробросить в роутере или включить UPnP.
• targetFrameRate: Фактический FPS сервера (рекомендуется ставить на 10-20 выше тикрейта).
• serverTickRate: Частота обновления сервера. Рекомендуется 240-360Hz+, если позволяет железо.
• clientTickRate: Частота отправки данных от клиентов (влияет на инпут-лаг). Можно ставить равной серверному тикрейту.
• Mods: В примере добавлены Toaster's Crispy Shadows (клиентский) и GAFURIX' Custom MOTD (серверный).

Шаг 8: Создание службы systemd

Создайте юнит для удобного управления сервером:

nano /etc/systemd/system/puck@.service

Вставьте следующее содержимое:

[Unit]
Description=Puck Dedicated Server - %i
After=network.target

[Service]
WorkingDirectory=/srv/puck/%i
User=puckd
ExecStart=/srv/puck/%i/start_server.sh --serverConfigurationPath /srv/puck/%i/config.json
Restart=on-failure

[Install]
WantedBy=multi-user.target

Шаг 9: Включение и запуск сервера

systemctl daemon-reload
systemctl enable puck@server1
systemctl start puck@server1

Шаг 10: Установка ufw (для безопасности)

apt install ufw
ufw enable
ufw allow 22/tcp
ufw allow 7777/udp
ufw allow 7778/udp
Примечание: Если у вас в конфиге сервера port и pingport не 7777 и 7778, то замените их на свои.

2. Скрипт объединения баз (merge_steams.sh)

cat << 'EOF' > /root/merge_steams.sh
#!/bin/bash
MASTER_DB="/srv/puck/server1/Logs/steam_db.csv"
INCOMING_DIR="/tmp/incoming_steam_data"
TMP_MERGE="/tmp/merged_db.csv"

# Собираем в кучу: текущую базу мастера + все присланные файлы от клиентов
cat "$MASTER_DB" "$INCOMING_DIR"/*.csv 2>/dev/null | awk -F';' '
BEGIN { OFS=";" }
{
    sid=$1; ips=$2; nicks=$3
    if (sid == "" || sid == "SteamID") next
    
    split(ips, ip_arr, ","); for (i in ip_arr) if (ip_arr[i] != "" && ip_arr[i] != "n/a") a_ips[sid][ip_arr[i]] = 1
    split(nicks, nick_arr, ","); for (i in nick_arr) if (nick_arr[i] != "" && nick_arr[i] != "n/a") a_nicks[sid][nick_arr[i]] = 1
}
END {
    print "SteamID;IP_Addresses;Nicknames"
    for (sid in a_ips) {
        ip_str = ""; for (ip in a_ips[sid]) ip_str = (ip_str=="" ? ip : ip_str "," ip)
        nick_str = ""; for (nick in a_nicks[sid]) nick_str = (nick_str=="" ? nick : nick_str "," nick)
        print sid, (ip_str=="" ? "n/a" : ip_str), (nick_str=="" ? "n/a" : nick_str)
    }
}' > "$TMP_MERGE"

if [ -s "$TMP_MERGE" ]; then
    mv "$TMP_MERGE" "$MASTER_DB"
    rm -f "$INCOMING_DIR"/*.csv
    echo "Слияние завершено успешно."
fi
EOF
chmod +x /root/merge_steams.sh

2. Парсер логов (update_steam_db.sh)

/srv/puck/server1/Logs/update_steam_db.sh 
cat << 'EOF' > /srv/puck/server1/Logs/update_steam_db.sh
#!/bin/bash

LOG_DIR="/srv/puck/server1/Logs"
DB_FILE="$LOG_DIR/steam_db.csv"
TMP_IPS="/tmp/steam_ips.txt"
TMP_NICKS="/tmp/steam_nicks.txt"

if [ ! -f "$DB_FILE" ]; then
    echo "SteamID;IP_Addresses;Nicknames" > "$DB_FILE"
fi

cat "$LOG_DIR"/ip_logger_*.ndjson 2>/dev/null | \
jq -R -r 'fromjson? | select(.steam_id != null and .ip != null) | "\(.steam_id)\t\(.ip)"' | \
awk '{
    split(a[$1], ips, ",")
    found = 0
    for (j in ips) if (ips[j] == $2) found = 1
    if (!found) a[$1] = a[$1] ? a[$1] "," $2 : $2
} END { for (k in a) print k "\t" a[k] }' > "$TMP_IPS"

gawk '
{
    sid=""; nick=""
    if (match($0, /Steam Id ([0-9]{17}).*\(([^)]+)\)/, m)) {
        sid = m[1]; nick = m[2]
    } 
    else if (match($0, /\[UIChat\] \[[^\]]+\] (.*) \([0-9]+\) \[([0-9]{17})\]:/, c)) {
        nick = c[1]; sid = c[2]
    }

    if (sid != "" && nick != "") {
        gsub(/^[ \t]+|[ \t]+$/, "", nick)
        if (nick ~ /^[0-9]{17}$/) next
        
        split(nicks[sid], ns, ",")
        found = 0
        for (i in ns) if (ns[i] == nick) found = 1
        if (!found) nicks[sid] = nicks[sid] ? nicks[sid] "," nick : nick
    }
} END { for (k in nicks) print k "\t" nicks[k] }' "$LOG_DIR"/Puck*.log 2>/dev/null > "$TMP_NICKS"

awk '
BEGIN { FS="[;\t]"; OFS=";" }
function merge(old, new,    arr, k, res, uniq) {
    split(old "," new, arr, ",")
    for (k in arr) if (arr[k] != "" && arr[k] != "n/a") uniq[arr[k]] = 1
    res = ""
    for (k in uniq) res = (res == "") ? k : res "," k
    return (res == "" ? "n/a" : res)
}
{
    if ($1 == "SteamID") next
    sid = $1
    if (FILENAME == db) { i[sid]=$2; n[sid]=$3 }
    else if (FILENAME == t_i) i[sid] = merge(i[sid], $2)
    else if (FILENAME == t_n) n[sid] = merge(n[sid], $2)
}
END {
    print "SteamID;IP_Addresses;Nicknames"
    for (sid in i) sids[sid]=1; for (sid in n) sids[sid]=1
    for (sid in sids) {
        print sid, (i[sid]?i[sid]:"n/a"), (n[sid]?n[sid]:"n/a")
    }
}' db="$DB_FILE" t_i="$TMP_IPS" t_n="$TMP_NICKS" "$DB_FILE" "$TMP_IPS" "$TMP_NICKS" > "${DB_FILE}.tmp"

mv "${DB_FILE}.tmp" "$DB_FILE"
rm -f "$TMP_IPS" "$TMP_NICKS"
EOF
chmod +x /srv/puck/server1/Logs/update_steam_db.sh
Эта вкладка позволяет превратить твой Главный сервер в веб-сервис, чтобы ты мог просматривать базу игроков прямо здесь, в браузере. Выполни эти 2 шага на Мастере.

1. Установка Python и запуск API на Мастере

apt update && apt install python3-pip -y
pip install fastapi uvicorn --break-system-packages

cat << 'EOF' > /root/puck_api.py
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware
import csv
import os

app = FastAPI()

app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],
    allow_methods=["*"],
    allow_headers=["*"],
)

DB_PATH = "/srv/puck/server1/Logs/steam_db.csv"

@app.get("/db")
def get_db():
    players = []
    if not os.path.exists(DB_PATH):
        return {"error": "File not found"}
    
    with open(DB_PATH, mode='r', encoding='utf-8') as f:
        reader = csv.DictReader(f, delimiter=';')
        for row in reader:
            players.append(row)
    return players
EOF

2. Автозапуск API (Systemd)

cat << 'EOF' > /etc/systemd/system/puck-api.service
[Unit]
Description=Puck Web API
After=network.target

[Service]
ExecStart=/usr/bin/python3 -m uvicorn puck_api:app --host 0.0.0.0 --port 8000
Restart=always
User=root
WorkingDirectory=/root

[Install]
WantedBy=multi-user.target
EOF

systemctl daemon-reload
systemctl enable puck-api
systemctl start puck-api
